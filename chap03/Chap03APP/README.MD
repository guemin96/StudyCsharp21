## 변수와 데이터 입력
- 핵심 포인트
  - 변수 선언으로 메모리에 저장 공간을 확보
  - 변수의 형태를 자료형이라 하며 정수형과 실수형으로 나눈다
  - 변수에 const를 사용하면 상수처럼 사용할 수 있다.
  - 예약어는 컴파일러와 약속된 단어이고, 식별자는 사용자가 만들어낸 단어이다.
